# 🍽️ Royale Palace

Royale Palace is an academic web application developed to streamline restaurant reservation management. The system allows administrators to manage reservations, menu items, users, and tables through an intuitive and responsive interface.

## 📖 Overview

This project was developed as part of the Software Development Engineering program at ITCA-FEPADE. It applies full-stack web development concepts using the Laravel framework and follows the MVC architecture.

## ✨ Features

- User authentication and authorization
- Restaurant reservation management
- Table management
- Menu management
- User administration
- Administrative dashboard
- CRUD operations
- Responsive interface
- Reservation reports

## 🚀 Installation

1. Clone the repository.

```bash
git clone https://github.com/karlarrivas/royale-palace.git
```

2. Install dependencies.

```bash
composer install
```

3. Copy the environment file.

```bash
cp .env.example .env
```

4. Generate the application key.

```bash
php artisan key:generate
```

5. Configure the database in the `.env` file.

6. Run the migrations.

```bash
php artisan migrate
```

7. Start the development server.

```bash
php artisan serve
```
